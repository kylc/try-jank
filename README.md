# Try jank

[![Static Badge](https://img.shields.io/badge/latest-blue?style=flat-square&logo=docker&label=ghcr.io%2Fkylc%2Ftry-jank)](https://github.com/kylc/try-jank/pkgs/container/try-jank)

Combines [jank](https://jank-lang.org/), [jank-nrepl-server](https://github.com/kylc/jank-nrepl-server), and [rebel-readline](https://github.com/bhauman/rebel-readline) into a plug-and-play Docker container.

``` sh
docker run -it --rm ghcr.io/kylc/try-jank:latest
```
